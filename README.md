## 🏛️ Technical Architecture

The system utilizes a high-performance **dual-stack approach** to maximize throughput while maintaining a modern user experience:

| Layer | Technology | Functional Role |
| :--- | :--- | :--- |
| **Backend Engine** | `.NET 9 / C#` | Handles low-level Steam protocols, manifest decryption, and multi-threaded content delivery. |
| **Frontend UI** | `CustomTkinter` | Provides a sleek, responsive dark-themed interface with dynamic scaling and real-time status. |
| **Automation** | `Batch Scripting` | Custom-built "Zero-Config" layer that auto-provisions the .NET SDK and Python environment. |

---

## 🛠️ Key Features

* **🚀 High-Speed Throughput:** Leverages .NET 9’s latest optimizations for maximum file download velocity.
* **🎨 Next-Gen UI/UX:** Modern, intuitive dark-themed interface designed for power users.
* **📦 Advanced Data Handling:** Custom VDF parsing engine combined with `ujson` for ultra-fast serialization.
* **⚙️ Automated Provisioning:** No manual path setup; the system automatically verifies and installs runtimes and libraries.
